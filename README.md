## ✏️리포지토리 제목

- 책 제목을 소문자와 하이픈(-)을 사용하여 작성
```java
sql-level-up
```
---
## 🗃️세션 자료 관리
- 각 챕터별로 폴더를 만들어 관리
- 1-9장은 01-09장으로 표시
- 위 리포지토리를 참고해 주세요
<br>

- 폴더 안에는 세션에 사용된 발표 자료를 md 파일로 저장
- 발표 자료 제목은 아래 양식을 따름 
- `{챕터}_{챕터 제목}_{발표자}`

```java
1장_DBMS 아키텍처_유건
```
<br>

- 한 챕터에 너무 많은 분량으로 챕터를 나눈 경우 아래와 같이 작성
```java
1-1장_DBMS 아키텍처 개요_유건
1-2장_DBMS와 버퍼_유진
```
